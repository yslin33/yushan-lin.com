# Chinese Writing Edition

## Editorial Principles

- Preserve the original text exactly.
- Never rewrite, paraphrase, shorten, or modernize the wording.
- Never change punctuation.
- Never change sentence order.
- Only adjust paragraph breaks when necessary for web readability.
- This is a web edition, not a rewritten edition.

## Typography

- Chinese serif: Noto Serif TC.
- Chinese body line-height: 1.58.
- Article title: weight 500.
- Body: weight 400.
- Semantic emphasis:
  - `<strong>` for original emphasized passages.
  - `<em>` for quotations and reflective passages.
- References use hanging indents.
